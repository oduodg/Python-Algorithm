## [완주하지 못한 선수](https://programmers.co.kr/learn/courses/30/lessons/42576)

- `collections.Counter()`

### Counter 합치기

```python
from collections import Counter
A = Counter({'a':1, 'b':2, 'c':3})
B = Counter({'b':3, 'c':4, 'd':5})
print(A + B)

👉🏻 Counter({'c': 7, 'b': 5, 'd': 5, 'a': 1})
```

### Counter 빼기

```python
from collections import Counter
A = Counter({'a':1, 'b':2, 'c':3})
B = Counter({'b':3, 'c':4, 'd':5})
print(B - A) # B에서 A와 중복되는 값을 빼줌

👉🏻 Counter({'d': 5, 'b': 1, 'c': 1})
```
